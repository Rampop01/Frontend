## Figma File
[Kolo Design - Figma](https://www.figma.com/design/YshYgkvgvjsxicYM4zY1Q0/Kolo-design?node-id=0-1&p=f&t=GmW9miz8bqfZIvLA-0)

## Overview
This submission provides the cohesive design system and high-fidelity screens for the Kolo MVP, reflecting Kolo's mission of making community finance simple, transparent, and secure.

## Deliverables Addressed

### 1. Landing Page
- Modern landing page with Hero section, Product overview, Features section, How Kolo Works, Stellar integration showcase, Community savings section, Testimonials, FAQ, and Footer.

### 2. User Dashboard
- Comprehensive dashboard displaying Wallet balance, Total savings, Active savings groups, Contribution statistics, Recent transactions, and Quick actions.

### 3. Savings Group Dashboard
- Group management experience including Group information, Savings progress, Contribution history, Member list, Upcoming payouts, and Invite member functionality.

### 4. Mobile Experience
- Mobile-first screens designed for User onboarding, Wallet balance, Savings group overview, Contribution confirmation, Notifications, and Payout confirmations.

### 5. Admin Dashboard
- Administrative dashboard displaying Platform metrics, User management, Active savings groups, Transaction analytics, Smart contract health indicators, and Activity logs.

## Design Alignment
- **Brand Personality:** Trustworthy, Modern, Community-Driven, Transparent, Friendly, Inclusive.
- **Colors:** Accurately utilized Primary Colors (Deep Navy `#0F172A`, Emerald Green `#10B981`, Stellar Blue `#2563EB`) and Secondary Colors (Teal `#0D9488`, Purple `#7C3AED`, Gold `#F59E0B`).
- **Typography:** Employed Space Grotesk for Headings and Inter for Body.
